## 📋 部署步骤

### 1. 数据库迁移
```bash
pnpm db:push
```

### 2. 配置环境变量
在 Vercel Dashboard 添加:
```bash
# 邮件服务
RESEND_API_KEY=re_xxx
RESEND_FROM_EMAIL=alerts@soloboard.com

# QStash 调度
QSTASH_TOKEN=xxx
QSTASH_CURRENT_SIGNING_KEY=xxx
QSTASH_NEXT_SIGNING_KEY=xxx
```

### 3. 创建 QStash 调度
```bash
pnpm tsx scripts/setup-qstash-alerts.ts
```

### 4. 测试
```bash
pnpm tsx scripts/test-alert-system.ts
```

## 📞 关键问题解答

### Q1: 如何导入流量数据？
进入站点详情 → 设置 → 配置 GA4 Property ID + Service Account

### Q2: 如何导入销售额？
进入站点详情 → 设置 → 配置 Stripe/Lemon Squeezy/Shopify API Key

### Q3: 删除网站按钮在哪？
Dashboard → 网站卡片 → 右侧三点菜单 → Delete

### Q4: 如何避免重复导入网站？
已自动实现，后端会检查域名重复

### Q5: 如何看到真实数据？
- Uptime 数据: 自动工作 ✅
- 收入数据: 需配置 API Key ⚠️
- 流量数据: 需配置 GA4 ⚠️

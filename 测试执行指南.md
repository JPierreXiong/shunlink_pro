### Phase 7: 报警系统 (15 分钟)

1. **创建报警规则**
   ```bash
   curl -X POST "http://localhost:3003/api/soloboard/alerts/rules" \
     -H "Content-Type: application/json" \
     -H "Cookie: your-session-cookie" \
     -d '{
       "siteId": "your-site-id",
       "type": "offline",
       "threshold": 0,
       "frequency": "immediate",
       "channels": ["email"]
     }'
   ```

2. **触发报警检查**
   ```bash
   curl -X GET "http://localhost:3003/api/cron/check-alerts?type=offline" \
     -H "Authorization: Bearer YOUR_CRON_SECRET"
   ```

3. **验证邮件**
   - [ ] 收到报警邮件
   - [ ] 邮件内容正确
   - [ ] 邮件格式美观
